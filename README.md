Call Taxi Booking Application

📌 Overview

The Call Taxi Booking Application is a simple console-based Java program that simulates how taxis are allocated to customers along a linear route. The system assigns taxis based on availability, proximity to the pickup point, and total earnings, ensuring efficient and fair allocation.

This project demonstrates core Java concepts such as Object-Oriented Programming (OOP), collections, sorting, and basic business logic implementation.
